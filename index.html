    Hugo #

    Hugo构建工具 #

    通过 Hugo 你可以快速搭建你的静态网站,比如博客系统,文档介绍,公司主页,产品介绍等等.

    hugo-book主题 #

    hugo-book 适合做文档功能的网页.Hugo众多主题中的一个

    演示 #

    • 主题演示地址 https://hugo-book-demo.netlify.app/

    安装 #

    Hugo安装 #

    • 最好使用对应版本，否则会有问题
    apt install -y hugo
wget gitdl.cn/github.com/gohugoio/hugo/releases/download/v0.124.1/hugo_extended_0.124.1_linux-amd64.tar.gz
tar -zxvf hugo_extended_0.124.1_linux-amd64.tar.gz -C /usr/bin/
rm hugo_extended_0.124.1_linux-amd64.tar.gz

    HugoBook使用 # 

    hugo new site mysite
cd mysite
wget gitdl.cn/github.com/alex-shpak/hugo-book/archive/refs/tags/v10.tar.gz
tar -zxvf v10.tar.gz -C themes
rm v10.tar.gz
cp -r themes/hugo-book-10/exampleSite/content.en/* content/
hugo server -p 80 --bind 0.0.0.0 --theme hugo-book-10
    • 版本 https://github.com/gohugoio/hugo/releases/tag/v0.124.1
    • 下载 hugo_extended_0.124.1_windows-amd64.zip解压放到C:\Windows
    • 主题 hugo-book.v10解压放到项目的themes目录下
    • hugo.toml配置文件如下:
    baseURL = 'https://example.org/'
defaultContentLanguage = 'zh'
#title = ''
theme = 'hugo-book-10'

[params]
  BookTheme = 'auto'
  BookDateFormat = '2006-01-02'
