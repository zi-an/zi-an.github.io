    禁止系统更新安全中心 #

    • https://www.sordum.org/9470/windows-update-blocker-v1-8/
    • https://www.sordum.org/files/downloads.php?st-windows-update-blocker

    安全中心通知 #

    gpedit.msc->计算机配置->管理模板->Windows 组件->Windows 安全中心->通知->隐藏所有通知

    修改缓存位置 #

    • 采用imdisk->设置->数据->从文件夹挂载
    • 文件夹里需要有\soft\edge和\soft\chrome
    del "C:\Users\docker\AppData\Local\Microsoft\Edge\User Data\Default\Cache"
mklink /J "C:\Users\docker\AppData\Local\Microsoft\Edge\User Data\Default\Cache" "Z:\soft\edge"


del "C:\Users\docker\AppData\Local\Google\Chrome\User Data\Default\Cache"
mklink /J "C:\Users\docker\AppData\Local\Google\Chrome\User Data\Default\Cache" "Z:\soft\chrome"
