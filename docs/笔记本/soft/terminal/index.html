    Windows Terminal #

    win+r 输入wt开启 #

    • win+shift+→ 把应用放到右扩展屏
    • win+↑ 最大化当前应用
    • ctrl+shift+t 打开新标签
    • ctrl+alt+1 打开第一个标签(配置json后改为ctrl+n)
    • shift+alt+ +/- 水平垂直拆分
    • shift+alt+ ↑↓←→ 水平垂直调整大小(需先拆分)

    win+1234 调出状态栏的第几个应用,无论时候打开
