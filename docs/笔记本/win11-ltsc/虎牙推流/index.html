    虎牙直播测试中

    • -re (input) 以本机帧速率读取输入。这相当于设置-readrate 1。
    • -stream_loop number (input) 设置输入流应循环的次数。Loop 0表示无循环，loop-1表示无限循环。
    • -codec[:stream_specifier] codec (input/output,per-stream)为一个或多个流选择编码器（当在输出文件之前使用时）或解码器（当在输入文件之前使用时）。codec是解码器/编码器的名称或特殊值copy（仅输出），指示流不被重新编码。
    • -c: copy 表示所有流格式都是复制
