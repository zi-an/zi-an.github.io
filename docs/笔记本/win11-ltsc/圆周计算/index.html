    计算圆周率 #

    • 圆周率查询 https://www.pilookup.com/zh-cn/
    • 圆周率下载 https://www.pilookup.com/zh-cn/download.html
    • 源代码 https://www.kurims.kyoto-u.ac.jp/~ooura/pi_fft.html
    • 最大ftt值为536870912大约512M内存

    使用代码计算 # 

    apt install gcc make -y

wget https://www.kurims.kyoto-u.ac.jp/~ooura/pi_fftc6_src.tgz
tar -zxvf pi_fftc6_src.tgz
cd pi_fftc6/
sed -i "s|-fast|-Ofast|g" Makefile_64bit
sed -i 's|scanf("%lg", &x)|x=100000|g' pi_fftcw.c
make -f Makefile_64bit

nohup ./pi_cw &
cat pi.dat
