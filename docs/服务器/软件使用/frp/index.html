    FRP内网穿透 #

    • https://github.com/fatedier/frp/
    • https://github.com/fatedier/frp/releases/download/v0.61.1/frp_0.61.1_linux_amd64.tar.gz

    服务端 # 

    wget https://github.com/fatedier/frp/releases/download/v0.61.1/frp_0.61.1_linux_amd64.tar.gz
tar -zxvf frp_0.61.1_linux_amd64.tar.gz --transform="s|frp_0.61.1_linux_amd64|frp|g"
./frp/frps -c ./frp/frps.toml

    客户端 #

    • 因服务端不在127.0.0.1,直接替换(未备案的域名可能被拦截无法访问,最好用IP)
    • 测试 ssh ali.mm -p 6000
    • 修改后缀.toml为.ini可以不输入参数
    sed -i 's|serverAddr = "127.0.0.1"|serverAddr = "ali.mm"|g' ./frp/frpc.toml
./frp/frpc -c ./frp/frpc.toml
